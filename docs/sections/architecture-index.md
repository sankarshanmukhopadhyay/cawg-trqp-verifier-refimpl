# Architecture & Deployment

This section covers how the verifier is built and how to run it: the control-plane
architecture, the optional trust gateway, the HTTP transport surface, deployment
modes (edge, gateway, service), operational hardening, and implementation-level
compatibility notes.

Start with **Architecture** for the system-level picture, then move to
**Deployment Guide** if you are standing up a service.
