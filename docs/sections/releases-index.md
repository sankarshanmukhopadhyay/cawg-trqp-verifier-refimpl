# Releases

The [Changelog](../../CHANGELOG.md) tracks unreleased and released changes at a
glance. Each tagged version below has its own release notes describing what
changed, what was fixed, and what compatibility impact (if any) downstream
consumers should expect.

Release notes are retained for every published version — nothing is pruned —
so that a downstream implementer pinned to an older version can still see
exactly what changed since the version they depend on.
