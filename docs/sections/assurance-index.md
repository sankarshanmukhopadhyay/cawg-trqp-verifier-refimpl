# Assurance & Evidence

This section is written for assurance reviewers, auditors, and conformance
programs. It covers decision receipts, audit bundles, reproducibility and
replay, the external assurance-suite ingestion manifest, the compatibility
matrix, the risk crosswalk, and release/handoff readiness.

Start with **How TRQP Enables Assurance** for the conceptual model, then
**Decision Receipt Specification** and **Audit Bundle Profile** for the
concrete artifacts.
