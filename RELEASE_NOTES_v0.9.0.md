# Release Notes v0.9.0

## Overview

v0.9.0 carries forward the v0.8.0 roadmap items and completes the v0.9.0 roadmap track. The repository now includes exportable audit bundles, HTTP transport patterns, benchmark fixtures, trust gateway mediation, richer interoperability vectors, and deployment guidance.

## Highlights

- audit bundle export for portable verification evidence
- trust gateway for remote policy mediation
- HTTP verify and audit bundle patterns
- benchmark fixtures for high-volume and constrained-device scenarios
- new deployment guide and non-technical overview

## Positioning

The repository now functions not only as a verifier reference implementation, but also as a compact blueprint for process-aware trust infrastructure.
