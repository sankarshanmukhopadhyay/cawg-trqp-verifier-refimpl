# Decision Receipt Examples

This directory contains example decision receipts for the three repository profiles:

- `decision_receipt_standard.json`
- `decision_receipt_high_assurance.json`
- `decision_receipt_edge.json`

Each file validates against `schemas/decision-receipt.schema.json` and illustrates how
the verifier can package input context, TRQP responses, policy evaluation, decision
outcome, and replay evidence as a machine-verifiable artifact.
